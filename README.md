# 城堡防守 (Castle Defense)

一款浏览器端的城堡防守小游戏。敌人从左右两侧进攻城堡，玩家通过布防士兵与施放技能来守卫城堡。

## 运行方式

```bash
npm install
npm run dev
```

浏览器打开终端显示的本地地址（如 `http://localhost:5173`）即可游玩。**支持手机浏览器**：用手机访问同一局域网下的地址即可触屏操作（点选技能后点击战场施放、点击城堡/士兵进行操作）。

## 构建

```bash
npm run build
```

产物在 `dist/` 目录，可部署到任意静态托管。

## 技术栈

- 纯前端：HTML + CSS + JavaScript (ES modules)
- 构建：Vite
- 渲染：Canvas 2D

## 游戏规则

- **目标**：守住城堡，别让敌人冲进城堡。
- **城堡血量**：开局为 10。每有一个敌人碰到城堡，血量减 1；血量归零即游戏结束。
- **波次**：敌人按波次出现。清完当前波所有敌人后，会提示「第 N 波 完成」，点击画面任意处开始下一波。下一波敌人更多、更强。
- **金币**：击败敌人获得金币，用来升级城堡、买士兵/炮兵、强化士兵、升级技能。

## 如何游玩

1. **防守**：己方士兵和城堡上的弓箭手会自动攻击靠近的敌人；升级城堡会增加弓箭手数量和防御。
2. **技能**：顶部有 9 个技能（火雨、火箭、火海、冻结、冰箭、冰环、导弹、弩炮、齐射）。先点选一个技能（或按数字键 1–9），再在战场上点击要施放的位置即可。每个技能有冷却时间。
3. **升级与增援**：
   - **点击城堡**：会弹出城堡面板，可在此升级城堡、恢复城堡血量、增加士兵、增加炮兵、以及强化全体士兵（攻击/生命/防御）。
   - **点击「商店」**：只用来给技能升级（每个技能最高 5 级）。
   - **点击己方士兵**：可查看属性，卫兵可花费金币升级为战士、弓箭手或圣骑士。
4. **天气**：每波结束后天气可能变化（晴/雨/雪/雷暴/冰雹等），部分技能会受影响（例如雨天火技能变弱、冰雹时机械类技能无法使用）。
5. **游戏结束**：城堡血量归零后可选择「重新开始」，或花费 300 金币恢复 5 点血量继续玩。
